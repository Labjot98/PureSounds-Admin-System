# 🎧 PureSounds Admin System

## 📌 Overview  
This project is a team-based admin system developed for the **PureSounds** headphone brand.  
It provides a centralized platform to manage core business operations, including customers, orders, stock, staff, and suppliers.

## 🚀 Features  
- Customer Management (Create, Read, Update, Delete)  
- Order Management  
- Stock Management  
- Staff Management  
- Supplier Management  
- Search and filtering functionality  
- Input validation for accurate data entry  
- Database integration using stored procedures  
- Secure handling of system data  

## 🛠️ Tech Stack  
- **Language:** C#  
- **Framework:** .NET (ASP.NET)  
- **Database:** SQL Server  
- **Concepts:** CRUD Operations, Validation, Stored Procedures
